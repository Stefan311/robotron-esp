Einleitung

Robotron-ESP ist eine Platine, mit der man das Video-Ausgangssignal von verschiedenen Robotron-Computern auf neueren Monitoren darstellen kann.
Zur Zeit wird die Ausgabe des Video-Signals über VGA und WLAN(HTTP-Streaming) unterstützt.
Das funktioniert aktuell für die Computer A7100, EC1834(mit MON-Grafikkarte), und PC1715.

Historie

Der Adapter wurde ursprünglich für den A7100 entwickelt, siehe Robotrontechnik-Forum.
Die Firmware stammt von Stefan Berndt (alias Ordoban), die Umsetzung auf Platine stammt von Mario Goegel (alias MarioG77).
Die Weiterentwicklung das Adapters kann auf Github verfolgt werden: Leiterplatte, VGA-Firmware und WLAN-Firmware.

Beschaffung

Mario hat einige Platienen fertigen lassen, und bietet auch die Bauteile als Set an. Einfach mal im Flohmarkt-Tread im Robotrontechnik-Forum fragen.
Alternativ dazu kann sich jeder den Schaltplan und Layout (KiCad) von Github herunterladen, und sich selbst Platinen fertigen lassen.
Die Bauteile könnt ihr dann zum Beispiel bei Reichelt kaufen. Hier die Materialliste. Auf Github sind auch die interaktiven Bestückungslisten für Hauptplatine und Keypad verfügbar.

Aufbauhinweise

Der Aufbau in SMD-Technik ist für ungeübte Finger schon eine Herausforderung.
Eine Anleitung mit vielen guten Tips von Mario kann helfen.

Was zu erwarten ist

Das Bild vom A7100 und EC1834MON sollte absolut klar und sauber sein.
Beim PC1715 haben wir hier leider kleine Qualitätseinbusen (zeitweise flackernde Pixel, insbesondere beim einschalten), die Bauartbedingt von der Hauptplatine kommen - vielleicht lässt sich da in Zukunft noch was verbessern. Hardwareseitig wird sich wohl nicht viel machen lassen.

Hinweise zum Flashen der Firmware

Die Firmware kann mit dem USB-Anschluss auf den Adapter übertragen werden. Wenn ihr nur die vorgefertigte Firmware nutzen wollt, dann empfehle ich die Benutzung der Flash Download Tools von Espressif.
Wer diese Methode nutzen möchte, sollte die detailierte Anleitung von Stefan lesen.

Wenn ihr jedoch selbst Änderungen an der Firmware vornehmen wollt, dann könnt ihr auch die Quelltexte von Github herunterladen, über die regulären Espressif Entwicklungsumgebungen (VS Code, Eclipse, ...) bearbeiten, compilieren, und auf den Adapter übertragen.
